ViiliSync

This application integrates a dynamic wagon weighing system with a wagon number recognition module and the ERP system.

The program communicates with the weight controller, collects weighing data, retrieves the corresponding wagon numbers, merges the information, and ensures secure local storage.

All processed data is automatically transferred to the ERP system, enabling complete and reliable tracking of wagon loadings.

Technologies Used
Delphi SQLite MS SQL Server Viinex
Key Features
  • Integration of dynamic weighing data and wagon number recognition
  • Direct communication with the weight controller
  • Automatic merging and validation of weighing results with wagon IDs
  • Local storage for reliability and backup
  • Automatic data transfer to the ERP system for further processing and reporting
Reliability & Recovery

Features advanced conflict resolution and automatic recovery mechanisms that ensure data integrity even in cases of network interruptions or system failures.
